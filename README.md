# reading-notes

一个轻量级、纯前端的阅读感悟记录工具，打开即用，无需安装任何软件。

## ✨ 功能简介
- 简易记录阅读心得、读书笔记
- 数据保存在浏览器本地，无需担心隐私泄露
- 界面简洁，专注于记录本身

## 🚀 使用方法（三步上手）
1.  **下载项目**
    点击仓库右上角的 `<> Code` → `Download ZIP`，下载并解压文件。

2.  **打开应用**
    找到解压后的文件夹，双击 `index.html` 文件，用浏览器打开即可。

3.  **开始记录**
    直接在页面上记录你的阅读感悟，关闭页面后内容会自动保存。

## 💡 注意事项
- 数据保存在浏览器本地，建议定期备份重要记录。
- 推荐使用 Chrome、Edge、Firefox 等现代浏览器打开，避免兼容性问题。
